# Azure-CLI-Automation-Project
**System Health Automation Script**

**Overview**

In this assignment, I developed a Bash script to automate the collection of system performance data on a Linux (Ubuntu) Virtual Machine.

**What I Accomplished:**

Scripting: Wrote a Bash script (collect_info.sh) that captures RAM usage, disk space, and system uptime.

Data Logging: Configured the script to output all data to a persistent log file (system_stats.log) with timestamps.

Permissions: Used chmod +x to make the script executable within the Linux environment.

Automation: Scheduled a Cron job to automatically run the script every minute, ensuring continuous system monitoring without manual intervention.

**Evidence**

The included screenshots document the entire process, from writing the code in the Nano editor to verifying the automated log entries.
